# TradeSense AI Pro

Trade Smarter. Get Funded.

TradeSense AI is a professional prop trading platform powered by AI insights. It helps traders navigate markets with confidence through AI signals, trade plans, risk detection, and smart sorting.

## Features

- **AI Trading Signals**: Buy, sell, and stop signals with confidence scores.
- **Prop Trading Challenges**: Prove your skills and get funded up to $100,000.
- **Leaderboard**: Compete with other traders and climb the ranks.
- **Community**: Connect with other traders in real-time.
- **MasterClass**: Learn from beginner to advanced levels.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```
